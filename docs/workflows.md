# Workflows

## Live probe workflow

1. identify exact route or tab
2. open in authenticated browser context
3. confirm ready state
4. capture runtime evidence
5. persist artifacts
6. state proof boundary

## Artifact build workflow

1. define source layer
2. define output grain
3. save raw artifact
4. save normalized derivative
5. update registry if artifact becomes current

## Normalization workflow

1. state raw field source
2. define normalization rule
3. classify proof level
4. persist normalized output
5. document limits and exclusions

## Excel build workflow

1. gather normalized inputs
2. separate forensic vs business sheets
3. add QC sheet when appropriate
4. preserve source and proof fields
5. save workbook under naming policy

## Delta pass workflow

1. read baseline artifacts
2. identify exact unresolved gap
3. run targeted continuation only
4. save `v2/v3/...` outputs
5. record current vs superseded relationship

## Targeted validation workflow

1. define disputed claim
2. choose narrow method
3. produce explicit evidence
4. classify result as proven / inferred / not_proven
