# Known Limits

## Popup linkage limits

- not every visit card yields a clean popup
- tooltip and popup layers must not be mixed
- stale popup carry-over must be treated as contamination

## Partial runtime evidence

- some fields remain structurally observed only
- some semantic closures are family-specific, not global

## Domain boundaries

- schedule runtime is not automatically a finance truth source
- patient-day enrichment is not always visit-level proof
- report export does not automatically equal canonical business truth

## Source limitations

- HTML-first screens may not expose dedicated API endpoints
- some flows are only partially observable through runtime

## Proof limitations

- inferred semantics must remain inferred
- narrow normalization must not be overgeneralized

## Global inventory limitations

- absence of a proven global source means claims must stay within observed scope
